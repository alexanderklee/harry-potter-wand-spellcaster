## Detailed Gesture Descriptions

### Alohomora
Draw a complete circle going **clockwise** (to the right at the top). Start at any point and return to your starting position.

### Revelio  
Draw a complete circle going **counter-clockwise** (to the left at the top). Opposite direction of Alohomora.

### Lumos
A quick, sharp upward flick. Start low, end high. Like you're turning on a light switch.

### Nox
The opposite of Lumos - a downward flick. Start high, end low.

### Incendio
A diagonal wave moving upward and to the right. Imagine drawing flames rising.

### Aguamenti
Draw an S-curve, moving generally downward. Like water flowing.

### Wingardium Leviosa
"Swish and flick!" - First sweep horizontally to the right, then flick upward at the end.

### Arresto Momentum
A horizontal sweep from left to right (or right to left). Smooth and controlled.

## Troubleshooting Recognition

| Issue | Solution |
|-------|----------|
| No detection | Check wand tip is visible, run calibration |
| Wrong spell | Make gestures more distinct, increase confidence threshold |
| Intermittent | Move slower, keep wand in frame throughout |
| Always same spell | Your gestures may look similar - exaggerate differences |
